# Face Detection using OpenCV

A simple Python project that detects human faces in real-time using OpenCV and a webcam.

## Technologies Used

* Python
* OpenCV

## Features

* Real-time face detection
* Webcam support
* Haar Cascade Classifier

## Run the Project

```bash
pip install opencv-python
python camera.py
```
